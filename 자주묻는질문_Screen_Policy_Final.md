# Screen Policy: 자주 묻는 질문 (FAQ)

## ■ Screen Meta
* **SCREEN_NAME**: 고객센터 > 자주 묻는 질문
* **화면 목적**: 고객이 자주 묻는 질문과 답변을 카테고리별로 조회
* **화면 형태**: 단일 페이지 (카테고리 탭 + 아코디언 목록)
* **진입 경로**: GNB > 고객센터 > 자주 묻는 질문
* **관련 화면**: 고객상담 화면, 마이페이지 > 문의내역 화면

## ■ 카테고리 탭 정책

### [탭 구성]
* **탭 항목**: A/S / 고객상담 / 제작 / 출고/배송 / 파일접수 / 회계관련
* **기본 선택**: A/S (첫 번째 탭)
* **탭 전환 시**: 해당 카테고리에 속한 FAQ 목록만 노출, 검색어 초기화

### [탭 카운트 뱃지]
* **표시 조건**: 검색어 입력 후 검색 실행 시, 해당 탭 내 검색 결과 수를 카운트 뱃지로 표시
* **표시 형식**: `A/S(0)`, `고객상담(2)` 형식
* **기본 상태 (검색 전)**: 카운트 뱃지 미표시
* **검색 후 상태**: 각 탭별 검색 결과 수 카운트 뱃지 표시
* **UI 정책**: 검색 결과가 있는 탭만 노출. 결과가 없는 탭은 화면에서 숨김 처리

---

## ■ 검색 정책

### [검색 필드]
* **필수 여부**: 선택
* **입력 규칙**: 텍스트 자유 입력
* **Placeholder**: "제목 또는 내용을 입력해주세요"
* **검색 실행 시점**: 돋보기 버튼 클릭 시 검색 실행
* **검색 범위**: 전체 카테고리에서 검색 후 결과 표시
* **검색 후 탭 동작**:
  - 전체 카테고리 검색 결과를 각 탭별로 분류하여 카운트 뱃지 표시
  - 카테고리 탭 선택 시 해당 탭의 검색 결과 목록 표시
* **결과 없음 시**: "검색 결과가 없어요." 안내 문구 표시

### [검색 결과 초기화]
* **초기화 조건**: 검색 필드 내용 삭제 후 재검색 또는 카테고리 탭 전환 시 검색어 초기화

## ■ FAQ 목록 정책

### [아코디언 동작]
* **기본 상태**: 모든 항목 닫힘 (답변 숨김)
* **펼침 동작**: 질문 클릭 시 답변 영역 펼침
* **복수 펼침**: 불가 — 다른 항목 클릭 시 이전에 펼쳐진 항목 자동 닫힘 (단일 펼침 방식)
* **닫힘 동작**: 펼쳐진 항목 재클릭 시 닫힘

### [Q (질문) 표시]
* **표시 항목**: "Q" 레이블 + 질문 텍스트
* **UI 정책**: 항목 우측에 펼침/닫힘 화살표 아이콘 표시

### [A (답변) 표시]
* **표시 항목**: 답변 텍스트 (서식 포함 가능: 줄바꿈, 강조 등)
* **노출 조건**: 질문 클릭(펼침) 시에만 노출
* **링크 포함**: 답변 내 링크 클릭 시 해당 화면으로 이동 (예: 마이페이지 문의내역)

### [목록 정렬]
* **정렬 기준**: 현업 확인 필요
* **페이징**: 무한 스크롤

## ■ 좌측 사이드바 네비게이션 정책

### [메뉴 구조]
* **대분류**: 이용 안내 / 배송 안내 / 회사 소개
* **소분류**: 각 대분류 하위 메뉴 (위 화면 구조 참조)
* **현재 페이지 강조**: 현재 선택된 메뉴 활성화 표시 (예: "자주 묻는 질문" 강조)

### [독립 상담 채널 링크] (회사 소개와 별도 그룹)
* **실시간 성원톡**: 클릭 시 실시간 채팅 채널로 이동
* **원격지원**: 클릭 시 원격지원 화면으로 이동
* **카카오톡 상담**: 클릭 시 카카오톡 채널로 이동

## ■ 예외 상황

### [검색 결과 없음]
* **처리**: "검색 결과가 없어요." 안내 문구 표시
* **표시 위치**: FAQ 목록 영역 내 중앙 표시

### [카테고리 내 검색 결과 0건]
* **처리**: 해당 탭 숨김 처리. 결과 있는 탭만 노출

### [네트워크 오류 시]
* **처리**: 에러 메시지 표시 및 재시도 유도

## ■ 필요 화면 목록

현재 화면 플로우에서 필요하지만 정의되지 않은 화면:

1. **마이페이지 > 문의내역 화면** - FAQ 답변 내 "마이페이지를 통해 문의 남겨주세요" 클릭 시 이동
2. **고객상담 화면** - 좌측 사이드바 고객상담 메뉴 클릭 시 이동
3. **FAQ 관리 화면 (관리자)** - FAQ 등록/수정/삭제 관리자 화면 (Q8 확인 후 범위 정의)
4. **실시간 성원톡 화면** - 사이드바 "실시간 성원톡" 클릭 시 이동
5. **원격지원 화면** - 사이드바 "원격지원" 클릭 시 이동
