# Screen Policy: 리뷰 관리

## ■ Screen Meta

* **SCREEN_NAME**: 리뷰 관리 (마이페이지 > 리뷰 관리)
* **화면 목적**: 배송 완료된 제품에 대한 리뷰 작성 및 작성한 리뷰 조회·수정·삭제
* **화면 형태**: 페이지 (탭 구조 + 다이얼로그 포함)
* **접근 대상**: 로그인 회원 전용
* **진입 경로**: 마이페이지 좌측 메뉴 > 활동 내역 > 리뷰 관리

---

## ■ 화면 구조

```
[리뷰 관리]
  ├─ [탭1] 작성 가능한 리뷰
  └─ [탭2] 작성한 리뷰
```

## ■ 공통 정책

### [리뷰 정책 요약]

| 항목 | 정책 |
|------|------|
| 작성 기한 | 배송 완료 후 1개월 이내 |
| 수정 | 자유롭게 가능 |
| 삭제 | 자유롭게 가능 |
| 삭제 후 재작성 | 불가 |

### [삭제 후 재작성 불가 정책 배경]

* **리뷰 신뢰도 보호**: 삭제 후 재작성을 허용하면 부정적 리뷰를 삭제하고 긍정적 리뷰로 대체하는 등 리뷰 내용을 사실상 소급하여 조작할 수 있어, 다른 고객의 구매 판단에 사용되는 리뷰의 신뢰성을 훼손할 수 있음
* **수정 기능으로 대체 가능**: 오타 수정, 내용 보완, 사진 교체, 별점 변경 등 재작성이 필요한 대부분의 케이스는 수정 기능으로 해결 가능하므로 사용자 불편 최소화
* **어뷰징 방지**: 삭제·재작성 반복을 통한 리뷰 내용 세탁, 별점 조작 등의 어뷰징 시도를 구조적으로 차단
* **국내 커머스 표준 준수**: 쿠팡, 네이버 스마트스토어, 올리브영 등 주요 플랫폼이 동일하게 재작성 불가 정책을 운영하고 있어 사용자 혼선 최소화

### [카테고리 필터]

* **탭 구성**: 전체 / 명함 / 홍보물 / 스티커 / 패키지·봉투 / 서식지 / 굿즈 / 기획제품
* **기본값**: 전체
* **UI 정책**: 탭 선택 시 해당 카테고리 항목만 필터링

### [목록 페이지네이션]

* **1페이지 노출 건수 기본값**: 20건
* **건수 변경**: Dropdown 선택 가능 (10건 / 20건 / 50건 / 100건)
* **정렬 기준**: 접수일 최신순

## ■ 탭1: 작성 가능한 리뷰

### [목록 표시 조건]

* **노출 조건**: 배송 완료 + 작성 기한(1개월) 이내 + 리뷰 미작성 건
* **기한 만료 처리**: 작성 기한 1개월 초과 시 목록에서 자동 제거
* **빈 상태**: "작성 가능한 리뷰가 없어요" 안내 문구 표시

### [리뷰 아이템 카드]

* **표시 항목**:
  * D-Day 뱃지 (예: D-1) — 작성 기한 만료 D-Day 표시
  * 접수명 (예: [디지털엽서] 김길동 고객 청접장)
  * 제품 옵션 요약 (용지 / 인쇄 / 규격 / 수량 / 가공)
  * 접수일 + 접수번호 (예: 2025-07-08 (sw12345678-01))
* **버튼**: [리뷰 작성]

### [D-Day 뱃지 표시 기준]

* **표시 시작**: 작성 기한 만료 D-30(30일 전)부터 표시
* **표시 형식**: D-30, D-29, ... D-1 순으로 카운트다운
* **미표시 조건**: D-30 이전 항목은 뱃지 미노출

## ■ 리뷰 작성 다이얼로그

### [진입 조건]

* 작성 가능한 리뷰 목록에서 [리뷰 작성] 버튼 클릭 시

### [화면 구성]

```
[리뷰 작성]

[제품 정보] (읽기 전용)
접수명 / 용지·인쇄·규격·수량·가공 옵션 요약

[별점] ★★★★★

[리뷰 내용 작성]
0/500

[사진 첨부] (선택)

[제품에 만족하셨나요?] — 만족스러운 점 선택 (필수)

[작성하기]
```

---

### [별점]

* **필수 여부**: 필수
* **입력 규칙**: 1~5점 (별 1개 단위 선택)
* **기본값**: 미선택 상태
* **UI 정책**:
  * 별 클릭 시 해당 점수로 선택
  * 미선택 시: [작성하기] 버튼 클릭 시 "별점을 선택해주세요" 에러 메시지 표시

---

### [리뷰 내용]

* **필수 여부**: 필수
* **입력 규칙**: 자유 텍스트
* **최소 글자 수**: 20자 이상
* **최대 글자 수**: 500자
* **UI 정책**:
  * 입력 글자 수 실시간 표시 (0/500)
  * 20자 미만 시: 글자 수 카운터를 빨간색으로 표시 (실시간 검증)
  * 안내 문구: "품질과 관계없는 사진이나 내용, 부적합한 내용 등은 삭제될 수 있어요"
  * 안내 문구: "다른 고객님에게 도움이 되도록 제품 후기를 공유해주세요 (최소 20자 이상)"
  * [작성하기] 버튼 클릭 시: 20자 미만이면 에러 메시지 표시 후 진행 차단
  * 에러 메시지: "최소 20자 이상 입력해주세요"

---

### [사진 첨부] (선택)

* **필수 여부**: 선택
* **허용 파일 형식**: JPG, PNG, GIF
* **개별 파일 크기 제한**: 10MB 이하
* **전체 합산 크기 제한**: 50MB 이하
* **최대 첨부 수**: 5장
* **UI 정책**:
  * 안내 문구: "사진은 JPG, PNG, GIF 파일로 최대 5장(개별 10MB, 전체 50MB 이하)까지 첨부할 수 있어요"
  * 5장 첨부 완료 시: [+] 추가 버튼 미노출
  * 허용 형식 외 파일 업로드 시: 즉시 에러 메시지 + 파일 자동 제거
    * 에러 메시지: "JPG, PNG, GIF 파일만 첨부할 수 있어요"
  * 개별 파일 크기 초과 시: 즉시 에러 메시지 + 파일 자동 제거
    * 에러 메시지: "파일 크기가 10MB를 초과했어요, 파일을 확인해주세요"
  * 전체 합산 크기 초과 시: 즉시 에러 메시지 + 파일 자동 제거
    * 에러 메시지: "첨부 파일의 전체 용량이 50MB를 초과했어요"

---

### [제품에 만족하셨나요? — 만족스러운 점 선택]

* **필수 여부**: 필수 (4개 카테고리 각 1개씩 모두 선택 필요)
* **선택 방식**: 카테고리별 단일 선택 (카테고리 내 1개만 선택 가능)
* **카테고리 및 항목**:

| 카테고리 | 선택 항목 |
|----------|-----------|
| 품질 | 원하는 색감이 잘 구현됐어요 / 재질이 만족스러워요 / 재단·가공이 깔끔해요 |
| 제작/접수 | 접수가 편리해요 / 오류없이 제작돼요 / 제작 속도가 빨라요 |
| 가격 | 가격 대비 품질이 좋아요 / 다른 곳보다 합리적인 가격이에요 / 재주문할 만큼 가성비가 좋아요 |
| 배송 | 배송이 빠르고 정확해요 / 포장이 꼼꼼하고 안전했어요 / 배송 현황 안내가 잘 돼요 |

* **UI 정책**: 각 카테고리 내 선택 항목 중 1개만 선택 가능, 다른 항목 클릭 시 기존 선택 해제 후 교체
* **미선택 시**: [작성하기] 버튼 클릭 시 "만족스러운 점을 모두 선택해주세요" 에러 메시지 표시 후 진행 차단

---

### [작성하기 버튼]

* **활성화 조건**: 항상 활성화
* **동작**: 필수 항목 유효성 검증 → 성공 시 리뷰 등록
* **유효성 검증 순서**:
  1. 별점 선택 여부 체크
  2. 리뷰 내용 최소 20자 이상 체크
  3. 만족스러운 점 4개 카테고리 전체 선택 여부 체크
* **성공 시**: 다이얼로그 닫힘 → 작성한 리뷰 탭으로 이동 및 새로고침
* **실패 시**: 해당 항목 에러 메시지 표시

### [닫기 버튼]

* **입력 데이터 없을 시**: 다이얼로그 즉시 닫힘
* **입력 데이터 있을 시**: 이탈 확인 다이얼로그 노출

  ```
  작성 중인 내용이 있어요
  페이지를 나가면 작성 중인 내용이 사라져요
  
  [계속 작성하기]  [나가기]
  ```

  * [계속 작성하기]: 이탈 확인 다이얼로그 닫힘, 리뷰 작성 다이얼로그 유지
  * [나가기]: 리뷰 작성 다이얼로그 닫힘, 입력 데이터 초기화

## ■ 현업 확인 필요사항

### Q1. 리뷰 수정 어뷰징 방지 정책

* **현재 상태**: 수정이 무제한으로 허용될 경우, 별점·내용을 반복 변경하는 방식의 어뷰징이 가능하여 추가 정책 검토 필요
* **확인 필요 사항**:
  1. 수정 횟수 제한 (예: 최대 1~2회)
  2. 수정 기한 제한 (예: 작성 후 N일 이내만 수정 가능)
  3. 별점 수정 불가, 텍스트·사진만 수정 허용
  4. 제한 없음 (현재 정책 유지)
  5. 현업 확인 필요
