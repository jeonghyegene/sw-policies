# Screen Policy: 고객센터 메인

## ■ Screen Meta

* **SCREEN_NAME**: 고객센터 메인
* **화면 목적**: 고객이 공지사항, 자주 묻는 질문, 자료실 등 고객지원 콘텐츠를 검색·탐색하고, 상담 채널에 접근하는 허브 화면
* **화면 형태**: 단일 페이지 (탭 구조 포함)
* **관련 화면**: 성원소식 목록/상세, 자주 묻는 질문 목록/상세, 자료실 목록/상세, 실시간 성원톡, 원격지원, 카카오톡 상담

---

## ■ 화면 구조

```
고객센터 메인
  ├─ [검색 영역] 검색어 입력 + 검색 결과 표시
  │     ├─ 검색결과 없을 때
  │     └─ 검색결과 있을 때 (성원소식 / 자주 묻는 질문 / 자료실 섹션별 표시)
  │
  ├─ [퀵 필터 탭] 가상계좌 / 교정 / 배송 / 주문 / 사이트 이용방법
  │
  ├─ [좌측 네비게이션]
  │     ├─ 성원소식
  │     ├─ 이용 안내
  │     ├─ 자주 묻는 질문
  │     ├─ 자료실
  │     └─ 오시는 길
  │
  ├─ [중앙 콘텐츠 영역]
  │     ├─ 최신 게시물 목록 (성원소식, 자주 묻는 질문 TOP 5, 자료실)
  │     └─ 서비스 바로가기
  │
  └─ [좌측 하단] 고객센터 / 실시간 성원톡 / 원격지원 / 카카오톡 상담
```

## ■ Field Rule

### [검색 영역]

#### [검색어 입력 필드]

* **필수 여부**: 선택
* **입력 규칙**: 자유 텍스트
* **UI 정책**:
  * Placeholder: "무엇이 궁금하신가요?"
  * Enter 또는 검색 아이콘 클릭 시 검색 실행
  * 검색어 입력 후 검색 실행 시 결과 영역으로 포커스 이동

#### [퀵 필터 탭]

* **필수 여부**: 선택
* **항목**: 가상계좌 / 교정 / 배송 / 주문 / 사이트 이용방법 (현업 확인 필요: 최종 항목 목록 및 순서 확정 필요)
* **관리 방식**: 하드코딩 (개발 배포를 통해서만 변경 가능)
* **UI 정책**:
  * 클릭 시 해당 키워드로 즉시 검색 실행
  * 선택된 탭 강조 표시
  * 검색어 입력 필드와 연동 (클릭 시 해당 키워드 자동 입력)

---

### [검색 결과 영역]

#### [검색결과 있을 때]

* **표시 항목**:
  * 검색어 표시: `"{검색어}"에 대한 검색결과`
  * 섹션별 결과 건수: 성원소식 (N) / 자주 묻는 질문 (N) / 자료실 (N)
  * 각 섹션별 결과 목록 표시
* **목록 표시 항목**: 제목 / 작성일 / 카테고리 태그
* **정렬 기준**: 최신순
* **최대 표시 건수**: 섹션별 20건, 이후 [더보기] 버튼 제공

#### [검색결과 없을 때]

* **표시 항목**:
  * 검색어 표시: `"{검색어}"에 대한 검색결과`
  * 섹션별 결과 건수: 성원소식 (0) / 자주 묻는 질문 (0) / 자료실 (0)
  * 안내 문구: "검색결과가 없습니다."
* **UI 정책**: 각 섹션 모두 0건으로 표시

---

### [좌측 네비게이션]

#### [메뉴 항목]

* **항목 목록**:
  * 성원소식
  * 이용 안내 (하위: 사이트 이용방법 / 배송비·운임표 / 배송 안내 / 묶음배송 / 낮배송/밤배송)
  * 자주 묻는 질문
  * 자료실
  * 오시는 길
* **UI 정책**: 현재 선택된 메뉴 강조 표시

---

### [콘텐츠 영역 - 메인 화면 (검색 미실행 상태)]

#### [최신 게시물 목록]

* **표시 항목**: 카테고리 태그 / 게시물 제목 / 작성일
* **표시 예시**:
  * `휴무공지 | 2025 여름휴가 업무일정 안내`
  * `AS | 택배가 파손되어 도착했어요`
  * `출고/배송 | 무인출고실에서 상품을 찾을 수 없어요!`
* **최대 표시 건수**: 5건
* **클릭 시**: 해당 게시물 상세 화면으로 이동

#### [자주 묻는 질문 TOP 5]

* **표시 항목**: 질문 제목 목록 (번호 없이 나열)
* **표시 건수**: 5건 고정
* **노출 기준**: 관리자 수동 지정
* **클릭 시**: 해당 FAQ 상세 화면으로 이동

#### [서비스 바로가기]

* **항목 구성**:
  * 사이트 이용방법
  * 배송비·운임표
  * 배송 안내
  * 묶음배송
  * 낮배송/밤배송
  * 회사 소개
  * 성원애드피아 Youtube
  * 특허 현황
* **클릭 시**: 해당 페이지로 이동

## ■ 필요 화면 목록

현재 화면 플로우에서 필요하지만 제공되지 않은 화면:

1. **성원소식 목록/상세 화면** - 성원소식 메뉴 클릭 또는 게시물 클릭 시 이동
2. **자주 묻는 질문 목록/상세 화면** - 자주 묻는 질문 메뉴 클릭 또는 FAQ 항목 클릭 시 이동
3. **자료실 목록/상세 화면** - 자료실 메뉴 클릭 또는 자료 항목 클릭 시 이동
4. **이용 안내 화면** - 사이트 이용방법 / 배송 안내 등 서브 메뉴 클릭 시 이동
5. **오시는 길 화면** - 오시는 길 메뉴 클릭 시 이동
6. **실시간 성원톡 화면** - 실시간 성원톡 클릭 시 노출
7. **원격지원 화면** - 원격지원 클릭 시 연결
