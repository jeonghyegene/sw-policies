# Screen Policy: 오시는 길

---

## ■ Screen Meta

* SCREEN_NAME: 오시는 길
* 화면 목적: 성원애드피아 매장(성수동·충무로·인현동)의 위치, 연락처, 교통편 안내

---

## ■ UI 구성 요소

### [탭 메뉴]
* 구성: 성수동 / 충무로 / 인현동
* 기본 선택: 성수동 (첫 진입 시)
* UI 정책: 선택된 탭은 활성 상태로 표시, 탭 전환 시 하단 콘텐츠 영역 교체
* 탭 간 정보 구성: 거의 동일하나 일부 항목 상이할 수 있음 → 현업 확인 필요사항 참조

### [지도 영역]
* 위치: 탭 콘텐츠 상단
* UI 정책: 해당 매장 위치를 지도로 표시

### [매장 정보 영역]
* 표시 항목:
  * 주소
  * 매장명
  * 매장 태그 (예: 공장 · 접수 · 출고 · 24시간 무인출고 운영)
  * 전화번호
  * 오시는 길 (텍스트 안내)
  * 지하철 이용 시
  * 버스 이용 시

### [전화번호]
* UI 정책:
  * PC: 클릭 시 동작 없음 (텍스트 표시만)
  * 모바일: 클릭 시 전화 앱 바로 연결

### [공유 버튼]
* UI 정책:
  * PC: 클릭 시 현재 페이지 URL 클립보드 복사 → "링크가 복사되었습니다" 토스트 메시지 표시
  * 모바일: 클릭 시 OS 기본 공유 시트 호출

---

## ■ 화면 결과

* 탭 전환 시: 선택한 매장의 지도 및 정보로 즉시 교체 표시
* 공유 버튼 클릭 시 (PC): 하단 토스트 메시지 "링크가 복사되었습니다" 노출 후 자동 소멸
* 공유 버튼 클릭 시 (모바일): OS 기본 공유 시트 호출

---

## ■ 예외 상황

* 지도 로딩 실패 시: 지도 영역에 "지도를 불러올 수 없습니다" 안내 문구 표시
* 모바일 공유 시트 미지원 환경: URL 클립보드 복사로 fallback 처리

---

## ■ 내부 확인 필요사항

* I1. 지도 서비스 종류
  * 선택된 상태: 내부 확인 필요 (카카오맵 / 네이버맵 / 구글맵 중 결정 필요)

* I2. 충무로·인현동 탭의 표시 항목 구성
  * 선택된 상태: 성수동과 거의 동일하나 일부 상이할 수 있음 → 탭별 표시 항목 및 태그 목록 확정 필요

---

## ■ 필요 화면 목록

* 충무로 탭 화면 - 매장 정보 및 태그 항목 확인 필요
* 인현동 탭 화면 - 매장 정보 및 태그 항목 확인 필요
