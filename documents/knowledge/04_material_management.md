# 자재 관리

## 1. 자재 분류 체계

| 분류 | 정의 | 예시 |
|------|------|------|
| **원자재** | 인쇄 생산의 주요 재료 | 용지(종이) |
| **부자재** | 생산에 필요한 보조 자재 | CTP 판, 잉크 |
| **소모품** | 사내 일반 소모품 | A4용지, 맥심, 커피 |

## 2. 원자재 관리

### 2.1 원자재 정보 관리
- 수정이력 내 **IP주소까지 표시**
- 등록 화면 **종목/형목 구분** 필요
- **재단유형 코드관리** (대표님 정리 후 제공 예정)
- **안전재고 항목 삭제** (재고현황 알림은 유지)

### 2.2 원자재 단가 등록

| 항목 | 내용 |
|------|------|
| 적용일 | 단가 **적용일** 포함 |
| 조정사유 | **에디터**로 변경 (첨부/에디팅 모두 필요) |
| 상세화면 | **단가 설정 상세화면** 필요 |
| 결재선 | **결재선** 필요 |

### 2.3 용어 변경
- 명칭 변경: 현재 최종가 → **현재 단가**
- 용어 변경: 직전 → **이전**
- 컬럼명 변경: 계약서 → **첨부파일**

## 3. 부자재 관리

### 3.1 부자재 정보 관리
- 부자재 분류 재확인 (기존 엑셀 참고하지 말 것)
- **계약서 파일등록 및 안전재고 설정 삭제**
- **대/중/소**만 구분
- **자재 이미지 등록** 필요

### 3.2 부자재 등록 화면
- 규격 → **단위**로 용어 변경

### 3.3 부자재 예시
- CTP 판
- 잉크
- 기타 생산 보조 자재

## 4. 소모품 관리

### 4.1 소모품 정의
- 사내에서 사용하는 일반 소모품
- 생산과 직접적인 관련 없음

### 4.2 소모품 예시
- A4용지
- 맥심
- 커피
- 기타 사무용품

## 5. 재고 관리

### 5.1 재고현황 알림
- 안전재고 항목은 삭제하되, **재고현황 알림은 유지**

### 5.2 구매팀 역할
- 자재, 부자재, 소모품에 대한 관리
- 발주 신청 처리

## 6. 단가 관리 프로세스

### 6.1 단가 등록 흐름
```
단가 정보 입력 → 조정사유 작성 (에디터/첨부) → 결재선 지정 → 승인 → 적용
```

### 6.2 단가 이력 관리
- 수정이력 내 IP주소 표시
- 이전 단가 이력 조회 가능
