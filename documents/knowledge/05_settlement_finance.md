# 정산 및 재경 정책

> **버전**: v1.6 | **최종 수정**: 2026년 2월 26일
> **변경 이력**: 세금계산서 발행 기준(결제일→출고완료), 현금영수증 발급 기준(배송완료→배송시작), 자진발급 방식, 성원머니 상세 운영 정책, S포인트 최소 사용 금액 추가

---

## 1. 세금계산서

### 1.1 발행 기준

#### 자동 발행
- **출고 완료 후, 익월 1일에 미발행 잔여 건 일괄 자동 발행**
- 대상: 수동으로 발행하지 않은 모든 건 (세금계산서 신청건 + 미신청건 포함)
- 자동 발행 후 수정 불가

#### 수동 발행
- 출고 완료 후 ~ 익월 1일 자동 발행 전까지 상시 신청 가능
- 방식: **건별 발행** 또는 **합산 발행** 선택 가능
- 합산 발행 기준: 사업자등록번호, 작성일은 선택 접수건 중 가장 늦은 날짜로 자동 설정

### 1.2 신청 가능 조건

| 구분 | 출고 전 | 출고 후 |
|------|---------|---------|
| 미신청건 | **신청 불가** | 익월 1일 전까지 신청 가능 (현금영수증 기신청건 제외) |
| 세금계산서 신청건 | 신청정보 수정 가능 | 신청정보 수정 가능 |

### 1.3 증빙 변경 규칙

| 변경 방향 | 배송 시작 전 | 배송 시작 후 |
|-----------|-------------|-------------|
| 세금계산서 → 현금영수증 | ✅ 가능 | ✅ 익월 1일 전까지 가능 / ❌ 수동 발행 완료된 경우 불가 |
| 현금영수증 → 세금계산서 | ✅ 가능 | ❌ **불가** |
| 미신청 → 세금계산서 | ✅ 가능 | ✅ 가능 (현금영수증 기신청건 제외) |

### 1.4 대리발급 정책
- 타 기업 정보로 세금계산서 발급 허용 (제3자 발행)
- 신청 시 기업 정보 직접 입력 또는 '나의 고객 목록'에서 선택

### 1.5 주의사항
- 발행 전 **폐업 여부 조회** 필수 (국세청 API 연동)
- 폐업 사업자: 수동 발행 시 발급실패 / 자동 발행 시 현금영수증 자진발급으로 자동 전환
- 부분 취소/추가 결제 발생 시 **기발급/미발급 금액** 관리 로직 명확화 필요
- ⚠️ **현업 확인 필요**: "배송시작일" vs "출고완료일" 용어 불일치 존재, 최종 기준 확정 필요

## 2. 현금영수증

### 2.1 발급 기준 시점
- **배송 시작 후**부터 발급/신청 가능

### 2.2 자동 발행 (회원 유형별 상이)

| 회원 유형 | 신청건 | 미신청건 |
|----------|--------|---------|
| **개인 회원** | 배송 시작 다음 날 00시에 입력된 번호로 자동 발행 | 배송 시작 다음 날 00시에 **국세청 지정번호(010-000-1234)로 자진발급** 자동 처리 |
| **기업 회원** | 배송 후에도 정보 수정 가능 | 익월 1일 자동 발행 전까지 신청 가능 |

> ⚠️ 개인 회원 자동 발행 후에는 정보 수정 및 취소 불가

### 2.3 신청 가능 기간
- 배송 시작 후 ~ 익월 1일까지

## 3. 환불/부분 취소

### 3.1 부분 취소 (교환/AS)
- **온라인 처리** 가능하도록 개편

### 3.2 카드 결제 환불
- 기존: 포인트로 환불 처리
- 변경 검토: **환불 계좌로 입금** 방식

### 3.3 가상 계좌
- 실제 구매자가 아닌 타인 입금 케이스 검토 필요
- 환불 계좌 변경 시 **사용자 본인 인증** 필수

## 4. S머니 (성원머니 / 예치금)

### 4.1 성격
- **예치금 성격** (환불성/충전성 분리)
- 통합 운영
- 환불 가능

### 4.2 거래 유형
- **입금**: 가상계좌 선입금 또는 접수 후 입금
- **사용**: 제품 접수 시 결제금액 차감
- **사용취소**: 전체/부분 취소 환불
- **출금**: 고객 요청에 의한 출금
- **소멸**: 소멸 시효 만료에 따른 예치금 소멸

### 4.3 가상계좌 정책
- 가상계좌 변경일로부터 **1개월간 재변경 불가**
- 지원 은행: 국민, 우리, 신한, IBK기업, NH농협, 수협, 하나은행 등

### 4.4 출금 정책
- **출금 가능 잔액**: 보유 잔액 - 출금대기 금액
- **출금 취소 가능 기간**: 신청 후 익일 오후 2시 30분까지
  - 금·토·일 신청 건은 월요일 오후 2시 30분에 처리
  - 취소 완료 시 즉시 출금 가능 잔액으로 복원
- 환불계좌 등록 필수 (계좌 인증 후 등록)

### 4.5 정책 결정 필요 사항
- ⚠️ 해외 금융기관 환불계좌 지원 여부 미확정
- ⚠️ 계좌인증 API 업체 미확정

## 5. S포인트 (리워드)

### 5.1 성격
- 리워드형 포인트
- **현금 환불 제외**

### 5.2 적립 기준
- **실 결제금액에 대해서만 적립** (포인트 사용분 제외)
- 추가 적립 이벤트 없을 예정

### 5.3 적립률 (등급별)

| 등급 | 적립률 |
|------|--------|
| 실버 | 1% |
| 골드 | 2% |
| 다이아몬드 | 4% |

### 5.4 사용 기준
- 포인트 소멸 우선 순위 적용
- 제품 취소 시 포인트 소멸
- **최소 사용 금액: 10,000P** (임시 확정, 오픈 전 최종 확정 필요)
- 세금계산서 발급 시 포인트 사용분은 발급 대상에서 제외 (실 결제금액 기준)
- 소멸 예정 포인트: 30일 이내 소멸 예정 포인트 및 소멸 예정일 안내

### 5.5 정책 결정 필요 사항
- 리워드 정책 변경에 따라 **정책 초안 작성**
- 담당 부서장과 협의 필요
