## 2. 회원 등급제

### 2.1 등급 체계 (6등급)

| 구분 | 등급명 | 기호 | 인정금액 기준 |
|------|--------|------|--------------|
| 일반 | 웰컴 (WELCOME) | W | 성원애드피아 신규 가입 고객 |
| 일반 | 진주 (PEARL) | P | 인정금액 500만원 이상 |
| 일반 | 브론즈 (BRONZE) | B | 인정금액 1,000만원 이상 |
| VIP | 실버 (SILVER) | S | 인정금액 3,000만원 이상 |
| VIP | 골드 (GOLD) | G | 인정금액 5,000만원 이상 |
| VIP | 다이아몬드 (DIAMOND) | D | 인정금액 1억원 이상 |

### 2.2 인정금액 산정 기준
- **상반기(1/1~6/30) / 하반기(7/1~12/31)** 이용 금액 기준으로 각각 산정
- 실 결제금액 기준 (성원포인트 사용분 제외)
- **제외 품목**: 전단 단/투명 하이브리드명함/메탈 스티커/재단형 등 일부 상품, 배송비
- ⚠️ 대상/제외 품목 전체 목록 현업 확인 필요

### 2.3 등급 조정(승급) 시점
- **매년 2회**: 2월 1일 / 7월 1일
- 등급 유지 기간: 산정일로부터 다음 산정일 전날까지
  - 예: 2월 1일 산정 등급 → 해당 연도 6월 30일까지 유지
  - 예: 7월 1일 산정 등급 → 해당 연도 12월 31일까지 유지

## 6. 탈퇴 처리

### 6.1 탈퇴 불가 조건
- **진행 중인 주문이 있는 경우 탈퇴 불가** (주문 취소 후 탈퇴 신청 필요)

### 6.2 탈퇴 시 소멸 항목
- 회원 등급
- 잔여 성원머니 (환불 불가)
- 잔여 S포인트 (환불 불가)

### 6.3 탈퇴 후 보관 항목
- **게시물·댓글 유지** (삭제 불가)
- 아이디(이메일), 비밀번호: 탈퇴 후 **7일 보관 후 삭제** (부정 이용 방지)
- 동일 아이디: 탈퇴 후 **7일간 재사용 불가**

### 6.4 법적 보관 의무 (전자상거래법)
| 기록 | 보관 기간 |
|------|----------|
| 대금결제 및 재화 공급 기록 | 5년 |
| 계약·청약철회 기록 | 5년 |
| 소비자 불만 및 분쟁처리 기록 | 3년 |

### 6.5 탈퇴 사유 선택 항목 (현업 확인 필요)
- 서비스 불만족 / 이용이 불편해서 / 개인정보 유출 우려 / 인쇄물 종류 부족
- ⚠️ 최종 항목 목록 확정 필요

## 7. 고객 관리 — 나의 고객 목록 (마이페이지)

### 7.1 기능 개요
- 마이페이지 내 기업 고객 목록 직접 관리 기능
- **세금계산서 대리발급 시 활용** (타 기업 정보로 발급 시 이 목록에서 선택)
- 접근: 마이페이지 좌측 메뉴 → 고객 관리

### 7.2 고객 목록 표시 항목
| 항목 | 설명 |
|------|------|
| 기업명 | 클릭 시 고객 정보 상세 다이얼로그 |
| 사업자등록번호 | 선택 입력 |
| 당해연도 접수건수 | 가입일 기준 1년 내 접수 건수 |
| 당해연도 총 결제금액 | 가입일 기준 1년 내 결제 금액 |
| 담당자명/연락처 | 선택 입력 |

### 7.3 고객 추가 정책
- **기업명 필수** (최대 30자, 중복 불가)
- 사업자등록번호 10자리 입력 완료 시 **휴/폐업 자동 조회**
- 동일 사업자등록번호 중복 등록 불가

### 7.4 고객 정보 상세 (기간별 집계)
- 기간 탭: 전체 / 최근 1개월 / 최근 6개월 / 최근 1년 / 기간 설정
- 기간 설정 시 당일 기준 최근 1년 이내 날짜만 선택 가능

### 7.5 삭제 정책
- 고객 정보만 삭제, 기존 접수 내역은 유지
- 삭제 후 복구 불가
