# 생산 프로세스

## 1. 생산 공정 흐름

```
조판 → 출력/인쇄 → 후가공/재단 → 포장 → 출고
```

## 2. 조판 관리

### 2.1 합판 기준
- **용지**: 동일 용지 기준
- **품목**: 동일 품목 기준
- **인쇄도수**: 동일 도수 기준
- **조판방법**: 양면/단면 구분
- **수량**: 수량 기준 선정

### 2.2 합판 예외
- 가공(코팅, 라미네이팅 등)은 **합판조건에 해당하지 않음**
- 단, 묶어서 가공 처리 시 **조판자 역량에 따라** 편리하게 처리 가능

### 2.3 조판 관리 상세

| 항목 | 상세 내용 |
|------|----------|
| **기기 배정** | 옵셋기기 배정 페이지에서 배정확정, 파일재배정, CTP 수량 조정 기능 활용 |
| **배송방법 표기** | 조판내역에서 동일배송끼리 묶음 필요 시 참고용으로 처리 |
| **자동조판** | 별사이즈도 독판 판단 시 자동 프로그램에서 조판 생성 |
| **조판메모/후가공/부가정보** | 각 항목별 사용 중, 데이터 필요 |

### 2.4 조판 승인
- **100만 원 이상 고액 접수건**에 한해 팀장 승인 필수
- 접수자 파일 조정 후, 조판 팀장이 진행 가능 여부 판단
- 승인 후 조판 진행

### 2.5 조판 특수 기능 (접수팀 논의 필요)

| 기능 | 설명 |
|------|------|
| **조판에러** | 접수에서 조판자에게 디자인 변경 등 사유로 조판 제외 요청 → 조판 에러 처리 후 응용프로그램에서 해당 조판번호 신규 파일로 업데이트 |
| **조판승인** | 고액 접수건 대상, 접수자 파일 조정 후 조판 팀장이 진행 가능 판단 시 승인 후 조판 진행 |

## 3. 출력/인쇄 프로세스

### 3.1 Prepress 프로세스
```
RIP(포맷변환/CMYK분리/각도포함) → CIP(인쇄가능파일 변환) → CTP(인쇄판)
```

### 3.2 파일 검증
- **PITSTOP (아그파)** 활용
- 오류 자동 검사 및 수정/검증

### 3.3 기기 연동
- 바코드 처리 후 자재 매칭
- 재단 관리 중요

### 3.4 생산 모니터링
- **RFID 연계**를 통한 소모품 관리
- 기기 오류 모니터링 기능 제공

## 4. 후가공/재단

### 4.1 재단 관리
- 재단 지시서 별도 제공
- 재단 가이드는 이미지 형태 유지

### 4.2 공정 표기
- 가공은 각 항목별로 사용
- 데이터 필요

### 4.3 진행 상태
- 생산 공정 중 담당자에게 메시지 발송
- **프로그레스 바**로 진행 상태 표시 예정

## 5. 포장
- 재단 완료 후 포장 공정 진행
- 출고 전 최종 단계

## 6. 출고/배송

### 6.1 배송 관리
- 택배비 로직 적용
- 제품 크기 및 무게에 따른 **자동 세팅 자료 수급** 필요

### 6.2 배송 완료 정의
- 출고 완료 후 특정 시간이 지난 시점으로 간주

### 6.3 월/주간 배송
- 월배송 및 주간배송에 대한 비용, 조건, 운영 시간대 등 세부 정책 확정 필요

## 7. 생산 자동화 연동

| 연동 | 기능 |
|------|------|
| **핫폴더 연동** | 조판 자동화, 규격/텍스트 검사 후 자동 인쇄 배정 |
| **RFID 연계** | 소모품 관리, 기기 오류 모니터링 |
| **PITSTOP** | 파일 오류 자동 검사/수정 |
