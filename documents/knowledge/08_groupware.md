# 그룹웨어

## 1. 개요
- 사내 업무 관리 시스템
- 성원애드피아 내부 직원용 시스템

## 2. GNB 메뉴 구성

| 메뉴 | 설명 |
|------|------|
| **홈** | 메인 대시보드 |
| **인사** | 인사 관리 |
| **근태** | 출퇴근/근태 관리 |
| **자산** | 회사 자산 관리 |
| **자재** | 자재 관리 |
| **회계** | 회계 관리 |
| **소모품** | 사내 소모품 관리 |
| **시설 및 차량** | 시설/차량 관리 |
| **재고관리** | 재고 현황 관리 |
| **전자결재** | 결재 프로세스 관리 |

## 3. 전자결재 (핵심 기능)

### 3.1 역할
- 그룹웨어의 **핵심 기능**
- 생산 과정에서 **승인이 필요한 프로세스**는 전자결재로 처리

### 3.2 전자결재 대상 프로세스

| 프로세스 | 결재 조건 |
|----------|----------|
| 조판 승인 | 100만 원 이상 고액 접수건 |
| 원자재 단가 등록 | 결재선 필요 |
| 기타 승인 필요 업무 | 각 업무별 결재선 지정 |

### 3.3 결재 흐름
```
[기안] → [결재선 지정] → [승인/반려] → [완료]
```

## 4. 인사 관리
- 직원 정보 관리
- 조직도 관리

## 5. 근태 관리
- 출퇴근 기록
- 휴가/연차 관리
- 근무시간 관리

## 6. 자산 관리
- 회사 자산 등록/관리
- 자산 현황 조회

## 7. 자재 관리 (그룹웨어 연동)
- 원자재/부자재/소모품 관리
- 구매팀 업무 연동

## 8. 회계 관리
- 재경팀 업무 지원
- 회계 데이터 관리

## 9. 소모품 관리
- 사내 소모품 관리
- A4용지, 커피 등 일반 소모품

## 10. 시설 및 차량 관리
- 시설 예약/관리
- 차량 배차/관리

## 11. 재고관리
- 원자재/부자재 재고 현황
- 재고 알림 기능

## 12. 생산 연계
- 생산에서 발생하는 승인 필요 업무는 **전자결재**로 처리
- ERP/MES 시스템과 연동
