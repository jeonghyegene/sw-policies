## 2. 자주 묻는 질문 (FAQ)

### 2.1 카테고리 탭
- 탭 항목: **A/S / 고객상담 / 제작 / 출고/배송 / 파일접수 / 회계관련**
- 기본 선택: A/S (첫 번째 탭)

### 2.2 검색 정책
- 전체 카테고리 검색 후 탭별 결과 분류 표시
- 탭에 검색 결과 수 카운트 뱃지 표시 (검색 후 결과 있는 탭만 노출)

### 2.3 아코디언 동작
- 기본: 모든 항목 닫힘
- 단일 펼침 방식: 다른 항목 클릭 시 이전 항목 자동 닫힘
- 페이징: 무한 스크롤

### 2.4 미결 이슈
- ⚠️ FAQ 목록 정렬 기준 확인 필요 (최신순 / 조회수순 / 수동 정렬)
- ⚠️ FAQ 등록/수정/삭제 관리 주체 및 권한 범위 정의 필요

## 5. 공지사항 (성원소식)

### 5.1 카테고리 체계

| 탭 (사용자 화면) | 포함 DB 분류 |
|----------------|------------|
| 전체 | 모든 분류 |
| 안내 | 휴무안내, 택배안내, 배송안내, 정책안내, 회계안내, 상품안내, 출고안내, 시스템안내 |
| 긴급공지 | 긴급공지 |
| 이벤트 | 이벤트 |
| 공모전 | 공모전 |
| 보도자료 | 보도자료 |
| 기타 | 기타 |

- 탭 구성: **하드코딩** (관리자 임의 변경 불가)
- 게시글 카테고리 뱃지: 탭명이 아닌 DB 분류명 그대로 표시 (예: 휴무안내, 출고안내)

### 5.2 공지 고정 게시글
- 관리자가 게시글 등록 시 **공지 여부 직접 지정**
- 공지 게시글은 목록 최상단 **항상 고정**, 복수일 경우 작성일 내림차순 정렬

### 5.3 게시글 기본 정책
- 정렬: 작성일 내림차순 (최신순), 페이지당 20건
- 검색: 제목+내용 통합 검색
- 첨부파일 다운로드: 비회원 포함 누구나 가능 (로그인 불필요)

## 6. 검색 화면

### 6.1 검색 기본 규칙
- 검색창 최대 30자, 특수문자 입력 불가
- 미입력 검색 실행 시 "검색어를 입력해 주세요." 안내

### 6.2 검색 초기 화면
| 영역 | 내용 |
|------|------|
| 최근 검색어 | 브라우저 로컬 저장, 로그인/비로그인 모두 저장, 최대 5개 |
| 인기 검색어 | 5개 노출 (판매량 기준) |
| 추천 키워드 | 관리자 등록 관리 방식 (내부 확인 필요) |

### 6.3 검색 결과 필터/정렬
- **카테고리 필터**: 결과에 포함된 카테고리만 동적 노출, 단일 선택
- **용지 필터**: 스탠다드/스페셜/패키지 구분, 다중 선택 (체크박스)
- **오늘판 필터**: 오늘판 신청 가능 제품만 필터링
- **정렬**: 인기순(기본, 판매량 기준) / 낮은가격순 / 높은가격순
- 검색 결과 없을 때: "검색결과가 없습니다." + 관리자 등록 추천 제품 노출

## 8. 리뷰 관리 (마이페이지)

### 8.1 기본 정책
| 항목 | 정책 |
|------|------|
| 작성 기한 | 배송 완료 후 **1개월 이내** |
| 수정 | 자유롭게 가능 |
| 삭제 | 자유롭게 가능 |
| 삭제 후 재작성 | **불가** (리뷰 신뢰도 보호, 어뷰징 방지) |

### 8.2 리뷰 작성 조건
- 탭 구조: [작성 가능한 리뷰] / [작성한 리뷰]
- 작성 가능 조건: 배송 완료 + 1개월 이내 + 리뷰 미작성 건
- D-Day 뱃지: 기한 만료 D-30부터 표시

### 8.3 리뷰 작성 필수 항목
- **별점**: 1~5점
- **리뷰 내용**: 최소 20자 ~ 최대 500자
- **만족스러운 점**: 품질/제작접수/가격/배송 4개 카테고리 각 1개씩 필수 선택
- **사진 첨부**: 선택 (JPG·PNG·GIF, 최대 5장, 개별 10MB, 전체 50MB)

### 8.4 카테고리 필터
- 전체 / 명함 / 홍보물 / 스티커 / 패키지·봉투 / 서식지 / 굿즈 / 기획제품

### 8.5 미결 이슈
- ⚠️ 리뷰 수정 어뷰징 방지 정책 확인 필요 (수정 횟수/기한 제한 여부)
