# Screen Policy: 공지사항 (성원소식)

## ■ Screen Meta
* **SCREEN_NAME**: 공지사항 (성원소식)
* **화면 목적**: 성원애드피아 운영 공지, 이벤트, 보도자료 등 소식 게시글 목록 조회 및 상세 확인
* **화면 형태**: 목록 페이지 + 상세 페이지
* **진입 경로**: GNB > 고객센터 > 성원소식

---

## ■ 화면 구조

```
공지사항 (성원소식)
  ├─ [카테고리 탭] 전체 / 안내 / 긴급공지 / 이벤트 / 기타 / 공모전 / 보도자료
  ├─ [게시글 목록]
  │   ├─ 공지 뱃지 + 카테고리 뱃지 + 제목 + 작성일
  │   └─ 페이징
  └─ [검색] 검색어 입력 + 검색 버튼
```

---

## ■ 목록 화면 Field Rule

### [카테고리 탭]
* **필수 여부**: 선택 (기본값: 전체)
* **탭 구성**: 전체 / 안내 / 긴급공지 / 이벤트 / 기타 / 공모전 / 보도자료
* **탭 구성 방식**: 하드코딩 (고정, 관리자 변경 불가)
* **탭-분류 매핑 기준**:

| 탭 | 포함 분류 (DB 카테고리) |
|----|----------------------|
| 전체 | 위 분류 전체 |
| 안내 | 휴무안내, 택배안내, 배송안내, 정책안내, 회계안내, 상품안내, 출고안내, 시스템안내 |
| 긴급공지 | 긴급공지 |
| 이벤트 | 이벤트 |
| 공모전 | 공모전 |
| 보도자료 | 보도자료 |
| 기타 | 기타 |

* **게시글 카테고리 뱃지**: 탭 명칭이 아닌 DB 분류명 그대로 표시 (예: 휴무안내, 출고안내, 긴급공지 등)
* **UI 정책**:
  - 탭 선택 시 해당 탭에 매핑된 분류의 게시글만 필터링하여 목록 갱신
  - 탭 전환 시 검색어 초기화 및 1페이지로 이동
  - 현재 선택된 탭 강조 표시

### [게시글 목록]
* **정렬 기준**: 작성일 내림차순 (최신순)
* **공지 뱃지 게시글 처리**:
  - 관리자가 게시글 등록 시 공지 여부 직접 지정
  - 공지 뱃지 게시글은 목록 최상단에 항상 고정 표시
  - 공지 고정 게시글이 복수일 경우 작성일 내림차순 정렬
  - 공지 게시글 이하 일반 게시글도 작성일 내림차순 정렬
* **페이지당 노출 건수**: 20건
* **표시 항목**:
  - 공지 뱃지 (공지 게시글에 한함)
  - 카테고리 뱃지 (예: 휴무안내, 출고안내, 정책안내 등)
  - 제목
  - 작성일 (예: 2025 07 14)
* **UI 정책**:
  - 게시글 제목 클릭 시 상세 화면으로 이동
  - 게시글이 없을 경우: "등록된 게시글이 없어요" 안내 문구 표시

### [검색]
* **필수 여부**: 선택
* **검색 범위**: 제목 + 내용 모두 검색
* **입력 규칙**: 자유 텍스트 입력
* **UI 정책**:
  - Placeholder: "제목 또는 내용을 입력해주세요"
  - 검색 버튼 클릭 또는 Enter 키 입력 시 검색 실행
  - 검색 결과 없음 시: "검색 결과가 없어요" 안내 문구 표시
  - 검색 실행 시 1페이지로 이동
  - 카테고리 탭 전환 시 검색어 초기화

### [페이징]
* **표시 방식**: 페이지 번호 버튼 (예: 1 2 3 4 5 6)
* **페이지당 노출 건수**: 20건
* **UI 정책**:
  - 현재 페이지 번호 강조 표시
  - 이전/다음 페이지 이동 버튼 제공

## ■ 상세 화면 Field Rule

### [게시글 상세 정보]
* **표시 항목**:
  - 공지/카테고리 뱃지
  - 제목
  - 작성일
  - 본문 (텍스트 및 이미지 포함)
  - 첨부파일 목록 (첨부파일이 있는 경우)
* **UI 정책**:
  - 본문 내 이미지 인라인 표시
  - 첨부파일 클릭 시 즉시 다운로드

### [첨부파일]
* **다운로드 권한**: 비회원 포함 누구나 다운로드 가능 (로그인 불필요)
* **표시 형식**: 파일명 + 다운로드 링크
* **UI 정책**:
  - 첨부파일이 없는 게시글은 첨부파일 영역 미노출

### [이전 / 다음 게시글 이동]
* **이동 기준**: 전체 게시글 기준 (카테고리 무관)
* **정렬 기준**: 작성일 기준
* **UI 정책**:
  - 이전 글 없을 경우: "이전 글이 없습니다" 표시 또는 비활성화
  - 다음 글 없을 경우: "다음 글이 없습니다" 표시 또는 비활성화

### [목록 버튼]
* **동작**: 공지사항 목록 화면으로 이동
* **이동 위치**: 이전에 조회하던 페이지 번호 유지 (세션 기반 복귀)

---

## ■ 화면 플로우

```
[GNB > 고객센터 > 성원소식]
  ↓
[공지사항 목록]
  ├─ 카테고리 탭 선택 → 필터링된 목록 표시
  ├─ 검색어 입력 → 제목+내용 검색 결과 표시
  ├─ 페이지 번호 클릭 → 해당 페이지 목록 표시
  └─ 게시글 클릭
        ↓
      [공지사항 상세]
        ├─ 첨부파일 클릭 → 파일 다운로드
        ├─ [이전] 클릭 → 전체 기준 이전 게시글 상세
        ├─ [다음] 클릭 → 전체 기준 다음 게시글 상세
        └─ [목록] 클릭 → 공지사항 목록 화면 이동
```

## ■ 예외 상황

### [검색 결과 없음]
* **처리**: "검색 결과가 없어요" 안내 문구 표시
* **재시도**: 검색어 수정 후 재검색 가능

### [게시글 없음]
* **처리**: "등록된 게시글이 없어요" 안내 문구 표시

### [첨부파일 다운로드 오류]
* **처리**: 에러 메시지 표시
* **재시도**: 사용자가 다시 클릭하여 재시도 가능

### [네트워크 오류]
* **처리**: 에러 메시지 표시 및 재시도 유도
