# 확인 필요사항 정리

해당 문서는 각 화면 정책서에 기재된 **내부 확인 필요사항** 및 **현업 확인 필요사항**을 취합한 것입니다.

## [검색_Screen_Policy_Final.md](./검색_Screen_Policy_Final.md)

### 내부 확인 필요사항

* **Q1. 용지 상세보기 모달 콘텐츠 관리 방식** ← 먼저 확인 필요
  - 모달에 노출되는 썸네일, 설명 등의 콘텐츠가 관리자 직접 등록 형태인지 확인 필요

* **Q2. 키워드 태그 운영 정책**
  - 제품당 등록 가능한 키워드 태그 개수 제한 여부 확인 필요
  - 등록된 태그가 5개 초과 시 노출 방식 확인 필요 (랜덤 5개 노출 / 등록 순서대로 5개 노출 / 전체 노출 등)

* **Q3. 추천 키워드 관리 방식**
  - 운영자 수동 등록/관리 여부 확인 필요

* **Q4. 인기 검색어 관리 방식**
  - 운영자 수동 등록/관리 여부 확인 필요

### 현업 확인 필요사항

* **Q1. 검색 필터 용지 항목 데이터**
  - 필터에 노출할 용지 전체 목록 및 분류(스탠다드/스페셜/패키지) 기준 데이터 수급 필요

  - 관리자 등록 형태로 확인 시, 현업 확인 필요사항 Q1(용지 항목 데이터 수급)은 불필요하므로 함께 검토

## [리뷰_관리_Screen_Policy.md](./리뷰_관리_Screen_Policy.md)

### 현업 확인 필요사항

### Q1. 리뷰 수정 어뷰징 방지 정책

* **현재 상태**: 수정이 무제한으로 허용될 경우, 별점·내용을 반복 변경하는 방식의 어뷰징이 가능하여 추가 정책 검토 필요
* **확인 필요 사항**:
  1. 수정 횟수 제한 (예: 최대 1~2회)
  2. 수정 기한 제한 (예: 작성 후 N일 이내만 수정 가능)
  3. 별점 수정 불가, 텍스트·사진만 수정 허용
  4. 제한 없음 (현재 정책 유지)
  5. 현업 확인 필요

### 내부 확인 필요사항

### Q1. 우편번호 검색 API 선택
* **현재 상태**: 내부 확인 필요
* **확인 필요 사항**:
  - 카카오 주소 API 사용

* **확인 필요 사항**:
  - 외국인 회원도 동일하게 국내 주소만 등록 가능
  - 외국인 회원은 해외 주소 등록 지원 필요 (국가 선택 + 해외 주소 입력 필드 별도 제공)
  - 외국인 회원의 휴대폰 번호 입력 규칙 (국내 번호만 vs 국제 번호 허용)

## [성원머니_통합_Screen_Policy.md](./성원머니_통합_Screen_Policy.md)

### 현업 확인 필요사항

### Q1. 해외 금융기관 추가 여부
* **현재 상태**: 현업 확인 필요
* **확인 필요 사항**:
  - 현재 환불계좌 변경 은행 목록은 국내 금융기관만 지원
  - 해외 금융기관(외국계 은행, 해외 계좌 등) 추가 지원 여부 결정 필요
  - 추가 시: 지원 국가 범위, 은행 목록, UI 노출 방식 (별도 탭 또는 기존 목록 통합) 정의 필요

### Q2. 계좌인증 API 업체 계약 확인
* **현재 상태**: 현업 확인 필요
* **확인 필요 사항**:
  - 환불계좌 등록·변경 시 사용되는 계좌인증 API 업체 미확정
  - 계약 완료 후 인증 실패 사유 코드 및 메시지 정의 필요
  - 업체별 지원 은행 범위가 다를 수 있으므로, 계약 후 지원 은행 목록 재검토 필요

## [세금계산서_발급_Screen_Policy_Final.md](./세금계산서_발급_Screen_Policy_Final.md)

### 현업 확인 필요사항

### Q1. 세금계산서 발급 신청 가능 시점 확인

* **현재 상태**: 현업 더블체크 필요
* **확인 필요 사항**: 세금계산서 신청 가능 시점 기준 불일치 — 아래 두 용어는 하루 이상 차이가 날 수 있으므로 어느 기준이 맞는지 확인 필요
  * **배송시작일**: 택배사가 물건을 픽업하여 실제 운송이 시작된 날 (피그마 화면 컬럼명, 증빙정책 원문 기준)
  * **출고완료일**: 성원애드피아 창고/공장에서 물건이 나간 날 (내부 처리 기준)
  * 현재 정책서는 '출고 완료' 기준으로 작성되어 있으나, 증빙정책 원문 및 피그마 화면은 '배송시작일' 기준으로 표기되어 있어 통일 필요
