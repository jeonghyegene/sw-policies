# Screen Policy: 자료실

## ■ Screen Meta
* **SCREEN_NAME**: 자료실
* **화면 목적**: 인쇄 작업에 필요한 템플릿, 서체 등 자료를 검색하고 다운로드하는 화면
* **화면 형태**: 단일 페이지 (목록형) + 상세 페이지
* **진입 경로**: 고객센터 → 자료실
* **관련 화면**: 고객센터 메인, 자료실 상세 화면

---

```
자료실

  │   ├─ 이용 안내

  │   │   ├─ 자주 묻는 질문

  │   │   ├─ 오시는 길
  │   │   └─ 사이트 이용방법
  │   ├─ 배송 안내
  │   │   ├─ 배송비·운임표
  │   │   ├─ 묶음배송
  │   │   └─ 낮배송/밤배송
  │   ├─ 회사 소개
  │   │   ├─ 성원애드피아 Youtube
  │   │   └─ 특허 현황
  │   └─ 그 외
  │       ├─ 원격지원
  │       ├─ 실시간 성원톡
```

---

## ■ Field Rule

---

### [필터 탭]

* **필수 여부**: 선택 (기본값: 전체)
* **탭 구성**: 전체 / 서체 자료실 / 템플릿 / 디자인
* **UI 정책**:
  - 탭 클릭 시 해당 카테고리 자료만 목록에 표시
  - 검색어 입력 후 탭 전환 시: 검색어 유지, 해당 탭 내 검색 결과 표시
  - 검색 결과가 존재하는 탭만 활성화하여 노출
  - 선택된 탭 강조 표시

---

### [검색어 입력 필드]

* **필수 여부**: 선택
* **입력 규칙**: 자유 텍스트 (제목 + 내용 통합 검색)
* **Placeholder**: "제목 또는 내용을 입력해주세요"
* **검색 실행 조건**: 검색 아이콘(🔍) 클릭 또는 Enter
* **UI 정책**:
  - 검색 실행 시 검색 결과가 있는 탭만 필터 탭에 노출
  - 검색 결과 없을 시 안내 문구 표시

---

### [자료 목록]

* **필수 여부**: 자동 표시
* **정렬 기준**: 공지 자료 상단 고정 → 이하 작성일 내림차순 (최신순)
* **페이지당 노출 건수**: 20건
* **표시 항목**:
  - 카테고리 뱃지 (예: 달력 템플릿, 캘린더 템플릿, 서체 자료실 등)
  - 자료명 (제목)
  - 작성일 (YYYY-MM-DD 형식)
* **UI 정책**:
  - 자료 클릭 시 상세 화면으로 이동
  - 공지 자료는 최신순 정렬과 별개로 목록 최상단에 고정 표시

---

### [페이지네이션]

* **표시 방식**: 숫자 페이지 버튼 (1 2 3 4 5 6 ...)
* **페이지당 노출 건수**: 20건

---

## ■ 자료실 상세 화면

### [화면 구성]
```
[자료명 (제목)]
작성일: YYYY MM DD
카테고리 뱃지

[내용 영역]
  - 텍스트 내용
  - 이미지 (선택)

첨부파일 [파일명.확장자]  [다운로드 버튼]

[이전] [다음] [목록] 버튼
```

---

### [자료명]
* **필수 여부**: 자동 표시 (수정 불가)
* **데이터 출처**: 자료 등록 시 입력된 제목

---

### [작성일]
* **필수 여부**: 자동 표시
* **표시 형식**: YYYY MM DD (예: 2025 07 14)

---

### [카테고리 뱃지]
* **필수 여부**: 자동 표시
* **표시 항목**: 공지 / 캘린더 템플릿 / 달력 템플릿 / 서체 자료실 / 디지털캘린더 / 옵셋캘린더 등

---

### [내용 영역]
* **필수 여부**: 자동 표시 (읽기 전용)
* **구성**: 텍스트 + 이미지 혼합 가능
* **UI 정책**: 스크롤로 전체 내용 열람 가능

---

### [첨부파일]
* **필수 여부**: 선택 (자료에 따라 첨부파일 없을 수 있음)
* **표시 항목**: 파일명 + 확장자
* **표시 예시**: `2024년 소량 벽걸이 캘린더 6종.pdf`
* **다운로드 동작**: 클릭 시 파일 즉시 다운로드 또는 저장 팝업 표시
* **로그인 필요 여부**: 로그인 없이 누구나 다운로드 가능 (비로그인 허용)
* **다운로드 횟수 제한**: 내부 확인 필요 → 별도 섹션 참조

---

## ■ Button Rule

### [이전 버튼]
* **위치**: 상세 화면 하단
* **동작**: 전체 목록 기준 이전 자료 상세 화면으로 이동
* **첫 번째 자료일 경우**: 버튼 미노출 처리 (이전 버튼 숨김)

### [다음 버튼]
* **위치**: 상세 화면 하단
* **동작**: 전체 목록 기준 다음 자료 상세 화면으로 이동
* **마지막 자료일 경우**: 버튼 미노출 처리 (다음 버튼 숨김)

### [목록 버튼]
* **위치**: 상세 화면 하단
* **동작**: 자료실 목록 화면으로 이동

## ■ 카테고리 구성

| 카테고리명 | 설명 |
|-----------|------|
| 전체 | 모든 자료 표시 |
| 서체 자료실 | 인쇄에 필요한 서체 통합 자료 |
| 템플릿 | 달력 템플릿, 캘린더 템플릿 등 |
| 디자인 | 디자인 관련 자료 |

---

## ■ 화면 플로우

```
[고객센터 메인]
  ↓ 자료실 메뉴 클릭
[자료실 목록 화면]
  ├─ 필터 탭 선택 → 카테고리별 자료 표시
  ├─ 검색어 입력 → 검색 결과 있는 탭만 활성화, 목록 표시
  └─ 자료 클릭
        ↓
     [자료실 상세 화면]
       ├─ 첨부파일 클릭 → 파일 다운로드 (비로그인 허용)
       ├─ [이전] → 전체 목록 기준 이전 자료 상세
       ├─ [다음] → 전체 목록 기준 다음 자료 상세
       └─ [목록] → 자료실 목록 화면
```

## ■ 예외 상황

### [검색 결과 없음 시]
* **처리**: "검색 결과가 없어요" 안내 문구 표시

### [자료 없음 시]
* **처리**: "등록된 자료가 없어요" 안내 문구 표시

### [첨부파일 다운로드 실패 시]
* **처리**: 에러 메시지 표시 및 재시도 유도

### [네트워크 오류 시]
* **처리**: 에러 메시지 표시 및 재시도 유도

---

## ■ 내부 확인 필요사항

### I1. 첨부파일 다운로드 횟수 제한 여부
* **확인 필요 사항**:
  1. 횟수 제한 없음 (무제한 다운로드)
  2. 회원당 N회 제한 (횟수 기준 내부 결정 필요)

## ■ 필요 화면 목록

현재 화면 플로우에서 필요하지만 제공되지 않은 화면:

1. **자료 등록/수정/삭제 화면 (관리자용)** — 관리자가 자료를 등록·수정·삭제하는 화면 필요
2. **로그인 다이얼로그** — 추후 로그인 필수 정책 변경 시 비로그인 사용자 접근 대응 필요
