# Screen Policy: 마이페이지 > 등급 안내

## ■ Screen Meta

* **SCREEN_NAME**: 마이페이지 > 등급 안내
* **화면 목적**: 회원의 현재 등급 현황 및 등급별 혜택 안내, 다음 등급 승급 조건 안내
* **화면 형태**: 단일 페이지
* **관련 화면**: 마이페이지 메인, 인정금액 안내 다이얼로그
* **접근 대상**: 로그인 회원 전용 (비로그인 사용자 접근 불가)

---

## ■ 화면 구조

```
[마이페이지 > 등급 안내]
  ├─ [상단] VIP 등급 혜택 안내 (실버 / 골드 / 다이아몬드)
  ├─ [중단] 일반 등급 혜택 안내 (웰컴 / 진주 / 브론즈)
  ├─ [하단] 등급별 현황 카드 (현재 로그인 회원 등급 기준)
  └─ [최하단] 승급 기준 안내 / 유의사항
```

| 구분 | 등급명 | 기호 | 인정금액 기준 |
|------|--------|------|--------------|
| 일반 | 웰컴 (WELCOME) | W | 성원애드피아 신규가입 고객 |
| 일반 | 진주 (PEARL) | P | 인정금액 500만원 이상 |

| VIP | 골드 (GOLD) | G | 인정금액 5,000만원 이상 |
| VIP | 다이아몬드 (DIAMOND) | D | 인정금액 1억원 이상 |

## ■ 등급별 현황 카드 Field Rule

### [공통 구성 요소]

#### [등급 진행 바]

* **표시 방식**: WELCOME → PEARL → BRONZE → SILVER → GOLD → DIAMOND 6단계 순서 표시
* **활성화 기준**: 현재 등급까지 진행된 형태 (현재 등급 강조)

#### [현재 등급 표시]

* **구성**: 등급 기호(알파벳) + "{회원명}님의 등급은 {등급명}이에요"

#### [2025년 인정금액]

* **표시 형식**: `{금액}원/{구매건수}건`
* **예외**: 웰컴 등급 → `0원/0건`

#### [승급 안내 문구]

* **표시 형식**: `{N}원만큼 추가 구매하시면 {다음등급명} 등급으로 승급해요`
* **금액 산출**: 다음 등급 인정금액 기준 - 현재 인정금액 (잔여 금액)
* **최상위 등급(다이아몬드)**: 승급 안내 문구 미표시, 축하 문구로 대체

#### [인정금액 안내 버튼]

* **위치**: 승급 안내 문구 우측
* **동작**: 인정금액 안내 다이얼로그 노출

## ■ 탭 구조 및 [받은 혜택] 탭 정책

### [탭 구조 적용 기준]

| 등급 | 탭 구조 | [받은 혜택] 탭 내용 |
|------|---------|-------------------|
| 웰컴 / 진주 / 브론즈 | 탭 없음 (단일 영역) | 해당 없음 |
| 실버 / 골드 / 다이아몬드 | [받은 혜택] / [등급 현황] 2탭 | 적립 포인트 표시 |

### [[받은 혜택] 탭 - 일반 등급 (웰컴/진주/브론즈)]

* **탭 미노출**: 탭 없이 현재 등급의 혜택 항목 목록을 단일 영역으로 표시

### [[받은 혜택] 탭 - VIP 등급 (실버/골드/다이아몬드)]

* **탭 노출**: [받은 혜택] / [등급 현황] 2탭 구조
* **[받은 혜택] 탭 표시 내용**: 적립 포인트
  * **표시 형식**: `적립 포인트 N,NNNP (최근 1년)`
  * **집계 기준**: 최근 1년

## ■ 승급 기준 안내 / 유의사항 (정적 콘텐츠)

### [인정금액 산정 기간]

* 멤버십 등급은 **상반기(1/1~6/30)** 와 **하반기(7/1~12/31)** 이용 금액 기준으로 산정

### [승급일]

* 매년 **2월 1일**과 **7월 1일**에 진행

### [등급 혜택 제공 기간]

* 확정된 멤버십 등급은 등급 산정일로부터 다음 등급 산정일 전날까지 유지
  * 예: 2월 1일 산정 등급 → 해당 해 6월 30일까지 유지
  * 예: 7월 1일 산정 등급 → 해당 해 12월 31일까지 유지

### [유의사항]

* 전단/투명 하이브리드명함/메탈 스티커/재단형 등 일부 상품, 배송비 등 결제 금액은 인정금액에서 제외
* 회원 등급 기준은 당사의 사정에 의해 변경될 수 있음
