# Screen Policy: 회원정보 수정

## ■ Screen Meta
* **SCREEN_NAME**: 회원정보 수정 (마이페이지)
* **화면 목적**: 가입된 회원 정보를 조회하고 수정
* **화면 형태**: 단일 페이지 (마이페이지 내)
* **진입 경로**: 마이페이지 > 내정보 관리 > 회원정보 수정
* **회원 유형별 구분**: 개인회원(본인인증) / 개인회원(SNS) / 외국인회원(본인인증) / 외국인회원(여권/이메일인증) / 기업회원(법인·개인사업자) / 기업회원(해외사업자)

---

## ■ 진입 전 보안 절차

### [보안 절차 노출 조건]
* **마이페이지 > 회원정보 수정 메뉴 클릭 시** 보안 절차 선행 실행

### [보안 절차 방식 - 회원 유형별]

| 회원 유형 | 보안 절차 |
|-----------|----------|
| 개인회원(본인인증) | 비밀번호 확인 |
| 개인회원(SNS 전용) | 본인인증 |
| 외국인회원(본인인증) | 비밀번호 확인 |
| 외국인회원(여권/이메일인증) | 비밀번호 확인 |
| 기업회원(법인·개인사업자) | 비밀번호 확인 |
| 기업회원(해외사업자) | 비밀번호 확인 |

### [비밀번호 확인 화면]
```
고객님의 소중한 정보 보호를 위해 비밀번호를 다시 한번 확인해 주세요.
비밀번호가 타인에게 노출되지 않도록 조심해 주세요.

[비밀번호 입력 필드]

[비밀번호 확인 버튼]
```
* **[비밀번호 확인] 버튼 동작**: 비밀번호 일치 검증 → 성공 시 회원정보 수정 화면 진입
* **실패 시**: 에러 메시지 표시, 재입력 가능

### [본인인증 화면 (SNS 전용 회원)]
```
고객님의 소중한 정보 보호를 위해 인증을 진행해주세요.
개인정보가 타인에게 노출되지 않도록 조심해 주세요.

[인증하기 버튼]
```
* **[인증하기] 버튼 동작**: 본인인증 API 팝업 호출
* **인증 완료 시**: 회원정보 수정 화면 진입


## ■ Field Rule (항목별)

### [비밀번호 변경]

#### [변경] 버튼 클릭 시 인라인 수정 UI 노출
```
비밀번호  [변경]
  ↓ 클릭
새 비밀번호     [**********]
* 영문/숫자/특수문자 중 2가지 이상을 포함하여 8~16자리로 입력해주세요   비밀번호가 일치합니다.
새 비밀번호 확인  [**********]  [변경]
```

* **새 비밀번호**
  - 입력 규칙: 영문/숫자/특수문자 중 2가지 이상 포함
  - 길이 제한: 8~16자
* **새 비밀번호 확인**
  - 입력 규칙: 새 비밀번호와 동일한 값
  - 일치 시: "비밀번호가 일치합니다." 안내 표시
* **[변경] 버튼**
  - 활성화 조건: 항상 활성화
  - 동작: 유효성 검증 → 성공 시 비밀번호 변경 완료 모달 표시

#### 비밀번호 변경 완료 모달
```
비밀번호 변경 완료
비밀번호 변경이 완료되었습니다.

[닫기]
```
* **[닫기] 버튼 동작**: 모달 닫힘, 회원정보 수정 화면 유지

### [이메일 주소 변경]

> **변경 방식은 회원 유형에 따라 상이**
> * 개인회원(본인인증), 외국인회원(본인인증): 중복 확인 후 변경 (이메일 인증 없음)
> * 개인회원(SNS), 외국인회원(여권/이메일인증), 기업회원(법인·개인사업자), 기업회원(해외사업자): 이메일 인증 후 변경

#### [이메일 주소 변경 모달 - 중복 확인 방식] (개인회원(본인인증), 외국인회원(본인인증))
```
이메일 주소 변경

이메일 주소
[이메일 주소를 입력해주세요]  [변경]
```

* **이메일 주소**
  - 필수 여부: 필수
  - 입력 규칙: 이메일 형식
  - 형식 검증 기준:
    - '@' 기호 1개만 포함
    - 처음/끝에 점(.), 하이픈(-) 입력 불가
    - 최소 하나 이상의 점(.) 포함
    - 점(.) 뒤 최상위 도메인은 2자 이상 영문
  - 미입력 시: "이메일 주소를 입력해주세요" 에러 표시
  - 형식 오류 시: "올바른 유형의 이메일 주소를 입력해주세요" 에러 표시
  - 중복 시: "이미 존재하는 이메일 주소에요" 에러 표시

* **[변경] 버튼**
  - 활성화 조건: 이메일 형식 유효성 통과 시
  - 동작: 중복 확인 → 성공 시 이메일 주소 즉시 변경 완료
